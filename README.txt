To use the Francis Cipher, run the main.py file. All available input variables will be able to be inputted when the program itself is run.

In case you would want to edit the Francis Cipher fixed key, please go into main.py and at line 10 change the key to 
whatever 32 bit hexidecimal after the |key = | in the form of 0xinsert32bithexhere.
